# 测试指南

## 🎯 当前已实现的功能

### 1. 统一结果面板 ✅
- **位置**: 所有Tab下方，共享使用
- **功能**:
  - 垂直拖拽调整高度
  - 折叠/展开按钮
  - 动态更新数据
  - 显示行数、列数、执行时间

### 2. 可视化查询Tab ✅
- **查询构建器**:
  - 左侧：查询模式卡片（字段选择、筛选、分组、排序、限制）
  - 右侧：动态内容区
  - 底部：生成SQL预览
- **功能**:
  - 点击模式卡片切换内容
  - 执行按钮触发查询
  - 结果显示在统一面板

### 3. 数据源面板 ✅
- **功能**:
  - 横向拖拽调整宽度
  - 折叠/展开按钮
  - 拖拽到50px以下自动折叠
  - 树形列表展开/折叠
  - 双击选择表

### 4. 其他Tab
- SQL查询、JOIN、SET、PIVOT（占位符，待完善）

## 🧪 测试步骤

### 测试拖拽功能
1. 打开 `docs/demo/index.html`
2. 拖拽数据源面板右侧边缘 → 应该能调整宽度
3. 拖拽到很窄 → 应该自动折叠
4. 拖拽统一结果面板上方分隔线 → 应该能调整高度

### 测试折叠功能
1. 点击数据源面板的折叠按钮 → 应该折叠/展开
2. 点击结果面板分隔线上的折叠按钮 → 应该折叠/展开
3. 折叠后再展开 → 应该恢复之前的尺寸

### 测试查询功能
1. 点击左侧不同的查询模式卡片 → 右侧内容应该切换
2. 点击"执行"按钮 → 结果面板应该显示数据
3. 切换到其他Tab → 结果面板应该保持显示

### 测试Tab切换
1. 点击不同的三级Tab → 内容应该切换
2. 结果面板应该始终可见（不随Tab切换而消失）
