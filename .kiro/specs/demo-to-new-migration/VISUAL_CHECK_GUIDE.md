# 视觉检查指南

## 🎯 快速检查步骤

### 步骤 1: 打开页面
1. 访问 http://localhost:5174
2. 点击左侧 Sidebar 的 "Query Workbench"（Code2 图标）

### 步骤 2: 检查布局
**应该看到三个区域**:
```
┌─────────┬─────────────┐
│         │             │
│  左侧   │    中间     │
│  面板   │    面板     │
│         │             │
│         ├─────────────┤
│         │   底部面板  │
└─────────┴─────────────┘
```

### 步骤 3: 检查左侧面板
**顶部**:
- [ ] 看到搜索框（带放大镜图标）

**中间**:
- [ ] 看到 "文件数据源" 分组（可点击展开/折叠）
- [ ] 看到 "数据库表" 分组（可点击展开/折叠）
- [ ] 如果有数据，看到表列表

**底部**:
- [ ] 看到两个按钮：刷新（圆圈箭头）和添加（+号）

### 步骤 4: 检查中间面板
**顶部 Tab 栏**:
- [ ] 看到 5 个 Tab：
  - SQL 查询
  - 表连接
  - 集合操作
  - 透视表
  - 可视化查询

**内容区域**:
- [ ] 看到占位文字（例如："SQL 查询编辑器（待实现）"）

### 步骤 5: 检查底部面板
- [ ] 看到占位内容
- [ ] 看到提示文字

### 步骤 6: 测试交互

#### 6.1 Tab 切换
1. 点击 "表连接" Tab
2. 内容应该变成 "JOIN 查询构建器（待实现）"
3. Tab 样式应该高亮

#### 6.2 表选择（如果有数据）
1. 在 "SQL 查询" Tab 下
2. 点击左侧的一个表
3. 表项应该有蓝色边框和浅蓝背景

#### 6.3 多选模式
1. 切换到 "表连接" Tab
2. 点击多个表
3. 应该可以同时选中多个

#### 6.4 右键菜单
1. 右键点击一个表
2. 应该弹出菜单：
   - 预览数据
   - 查看结构
   - 删除表（红色）

#### 6.5 搜索功能
1. 在搜索框输入表名的一部分
2. 表列表应该实时过滤

#### 6.6 调整布局
1. 拖动左侧和中间之间的分隔线
2. 应该可以调整宽度
3. 分隔线悬停时应该变蓝色

#### 6.7 主题切换
1. 点击右上角的主题切换按钮
2. 切换到深色模式
3. 所有颜色应该正确切换

## 🐛 常见问题

### 问题 1: 左侧面板是空的
**原因**: 没有数据源  
**解决**: 
1. 点击左下角的 "+" 按钮
2. 或切换到 "数据源管理" 页面上传数据

### 问题 2: 看不到 Query Workbench 导航项
**原因**: 可能在旧版本布局  
**解决**: 
1. 检查 localStorage 中的 `dq-use-new-layout`
2. 确保使用新布局

### 问题 3: 样式不正确
**原因**: 可能是缓存问题  
**解决**: 
1. 硬刷新浏览器（Ctrl+Shift+R 或 Cmd+Shift+R）
2. 清除浏览器缓存

### 问题 4: Tab 切换没反应
**原因**: JavaScript 错误  
**解决**: 
1. 打开浏览器控制台（F12）
2. 查看是否有错误信息
3. 刷新页面
