## 🔍 如果看不到内容

### 检查 1: 是否有数据源
如果左侧面板是空的，说明还没有数据源。需要：
1. 点击左下角的 "+" 按钮
2. 或者切换到 "数据源管理" 页面
3. 上传文件或连接数据库

### 检查 2: 是否在正确的页面
确保：
1. 左侧 Sidebar 中 "Query Workbench" 是激活状态
2. 不是在 "统一查询" 或其他页面

### 检查 3: 浏览器控制台
打开浏览器开发者工具（F12），检查：
1. Console 是否有错误
2. Network 是否有请求失败

### 检查 4: 编译状态
终端中应该显示：
```
✓ built in XXXms
```
如果有错误，会显示红色错误信息。
